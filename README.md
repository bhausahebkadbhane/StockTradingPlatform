## 📌 Project Description

This project is a **console-based Stock Trading Platform** developed using Java.  
It simulates a basic stock market where users can:

- Buy stocks 📥  
- Sell stocks 📤  
- Track portfolio 📊  
- View transaction history 🧾  

It also includes **data persistence**, so user data is saved and loaded from files.

## ⚙️ How It Works

### 1. User Initialization
- User starts with **$10,000 balance**

### 2. Market System
- Predefined stocks:
  - AAPL
  - GOOGL
  - MSFT
  - TSLA

### 3. Buy Stock
- User enters stock symbol & quantity
- Balance gets deducted
- Portfolio updated

### 4. Sell Stock
- User sells owned stocks
- Balance increases

### 5. Portfolio
- Shows:
  - Stocks owned
  - Quantity
  - Current value

### 6. Transactions
- Every buy/sell is recorded with timestamp

### 7. Data Storage
- Saved in:
  - `user_data.txt`
  - `transactions.txt`
